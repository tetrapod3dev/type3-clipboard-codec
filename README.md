# TYPE3 Clipboard Codec

TYPE3 Edit 프로그램에서 복사된 이진 클립보드 데이터를 역공학하고, 파싱하고, 향후 재생성까지 지원하기 위한 파이썬 라이브러리입니다.

이 프로젝트는 TYPE3 Edit에서 객체를 복사했을 때 생성되는 독자적인 바이너리 클립보드 포맷을 분석하는 데 목적이 있습니다.  
현재는 포맷을 안정적으로 관찰하고 구조를 문서화하며, 일부 객체 타입에 대해 파싱 가능한 형태로 정리하는 데 초점을 두고 있습니다.  
장기적으로는 파싱된 객체를 수정한 뒤 다시 바이너리로 인코딩하여, 프로그램적으로 클립보드에 주입하고 붙여넣기하는 흐름까지 지원하는 것을 목표로 합니다.

## 프로젝트 목적

- TYPE3 Edit의 독자적인 이진 클립보드 포맷 분석
- 객체 타입 식별 및 구조 해석
- CLI를 통한 원시 데이터 미리보기 및 검사
- 객체 수정 및 재인코딩 지원을 위한 기반 마련
- 향후 클립보드 주입 및 자동화 워크플로우 지원

## 현재 중점 대상

현재까지 중점적으로 다루고 있거나 분석 대상으로 삼고 있는 객체는 다음과 같습니다.

- Rectangle
- Line
- Arc
- Circle
- Rounded Rectangle
- Text

일부 객체는 구조가 부분적으로만 해석되어 있을 수 있으며, 포맷 이해가 진행됨에 따라 표현 방식이나 필드 해석 결과가 바뀔 수 있습니다.

## 핵심 설계 원칙

- **Codec 중심 설계**  
  단순한 일회성 파서가 아니라, 디코딩 → 수정 → 인코딩 흐름까지 확장 가능한 코덱 구조를 지향합니다.

- **관심사 분리**  
  데이터 입력 소스(클립보드, 파일, Hex 텍스트 등)와 실제 코덱 로직을 어댑터 패턴으로 분리합니다.

- **확장성**  
  새로운 객체 타입이 발견될 때마다 독립적인 파서와 감지기를 추가할 수 있는 구조를 지향합니다.

- **역공학 친화성**  
  불확실한 필드는 억지 해석하지 않고 원시 값, 후보 값, 미확정 필드로 보존하여 정보 손실을 줄입니다.

- **점진적 정밀화**  
  처음부터 완전한 모델을 강제하지 않고, 샘플 축적과 비교 분석을 통해 점진적으로 필드 의미를 확정합니다.

## 프로젝트 구조

```text
src/type3_clipboard_codec/
├─ adapters/   # 다양한 입력 소스 처리 (Hex 텍스트, Win32 클립보드 등)
├─ codec/      # 디코더, 인코더, 미리보기 렌더러
├─ parsers/    # 객체별 파싱 로직 및 감지기
├─ models/     # 파싱 결과를 표현하는 정형 모델
└─ utils/      # 바이너리 읽기, 변환, 공통 유틸리티

tools/
└─ inspect_manual_hex.py  # 수동 Hex 입력 검사 도구

docs/
├─ format_notes.md
└─ reverse_engineering_log.md
````

## 설치 및 사용법

현재는 개발 중인 프로젝트이며, 소스 코드를 직접 체크아웃하여 사용하는 형태를 기준으로 합니다.

```bash
pip install -e ".[dev]"
```

수동 Hex 입력 검사 도구 실행:

```bash
python tools/inspect_manual_hex.py
```

## 현재 상태

이 프로젝트는 **실험적(experimental)** 이며, 관찰된 샘플 데이터를 바탕으로 포맷을 역공학하는 단계에 있습니다.
따라서 일부 객체 타입이나 필드에 대한 해석은 아직 불완전할 수 있으며, 새로운 샘플 분석 결과에 따라 구조나 해석 방식이 변경될 수 있습니다.

## 향후 계획

1. 객체별 상세 파서 구현 고도화
2. 파싱된 모델의 속성 수정 지원
3. 수정된 모델을 다시 이진 데이터로 변환하는 인코더 구현
4. Windows 클립보드 직접 연동 어댑터 고도화
5. 샘플 기반 회귀 테스트 및 round-trip 검증 체계 마련

## 주의사항

이 프로젝트는 TYPE3 Edit의 비공식 포맷을 분석하는 목적으로 작성되었습니다.
공식 문서에 기반한 구현이 아니므로, TYPE3 버전 차이 또는 환경 차이에 따라 클립보드 구조가 달라질 가능성이 있습니다.
